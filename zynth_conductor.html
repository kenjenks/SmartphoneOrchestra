🎯 Pattern Encoding

Each note contains multiple warbles that sound like flute harmonics or room echoes. The pattern (101, 1001, etc.) is encoded in the timing of the warbles.

  • Pure tone - No warbles (sounds like held flute note)
  • 101 - Warble, pause, warble (sounds like double echo)
  • 1001 - Warble, long pause, warble (sounds like distant echo)
  • 10101 - Five rapid warbles (sounds like harmonic series)

⚡ Room noise might trigger one random warble, but not a specific pattern!
